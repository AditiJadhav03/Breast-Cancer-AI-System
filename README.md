# 🧠 Breast Cancer Treatment Recommendation System (AI-Powered)

An end-to-end Machine Learning + Deep Learning + Streamlit web application that predicts the most suitable breast cancer treatment based on clinical patient data.

The system recommends:
- 💊 Chemotherapy  
- 🧬 Hormonal Therapy  
- 🔬 Surgery

## 📌 Project Overview

This project is a clinical decision support system built using machine learning models trained on breast cancer patient data.

It predicts treatment based on:

- Age  
- ER / PR / HER2 status  
- Cancer stage  
- Tumor characteristics  
- Lymph node count  
- Histology type  

---

## 🧠 Machine Learning Models

- 🌲 Random Forest Classifier (Main Model)
- 🧠 CNN Model (Experimental)
- 🔁 LSTM Model (Experimental)

👉 Final deployment uses: **Random Forest (Best Accuracy)**

---

## 📊 Model Performance

- Accuracy: ~94–95%
- Multi-class classification:
  - Chemotherapy
  - Hormonal Therapy
  - Surgery
